# Personnel Management System

This repository contains a personnel management system that allows you to manage personnel data, including adding, editing, and deleting personnel records.

## Project Setup

To set up the project, follow these steps:

1. Clone the repository:
   ```bash
   git clone https://github.com/yanqianyu/personnel-management.git
   cd personnel-management
   ```

2. Install the dependencies:
   ```bash
   npm install
   ```

3. Run the application:
   ```bash
   npm start
   ```

## Features

- Add new personnel records
- Edit existing personnel records
- Delete personnel records
- View a list of all personnel
